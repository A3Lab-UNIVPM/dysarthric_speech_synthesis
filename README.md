# Improving Dysarthric Speech Classification with Generative AI-based Data Augmentation

This repository contains the project files for the generative AI-based data augmentation framework and the downstream intelligibility classification system described in:

> Lillini D., Basili T., Migliorelli L., Aironi C., Gabrielli L., Squartini S.
> *Improving Dysarthric Speech Classification with Generative AI-based Data Augmentation*

The [demo page](https://a3lab-univpm.github.io/dysarthric_speech_synthesis/) presents audio samples synthesized by the Grad-TTS model as part of the generative data augmentation framework described in the paper.
Samples include original microphone recordings, synthesized speech on TORGO transcriptions, and synthesized speech on UA-Speech vocabulary words.
